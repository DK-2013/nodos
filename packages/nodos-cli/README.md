# Usage
<!-- usage -->
```sh-session
$ npm install -g @nodos/cli
$ nodos COMMAND
running command...
$ nodos (-v|--version|version)
@nodos/cli/0.0.0 darwin-x64 node-v10.10.0
$ nodos --help [COMMAND]
USAGE
  $ nodos COMMAND
...
```
