# Nodos

Node.js framework for humans

### Install

```sh
$ npm i -g @nodos/cli
$ nodos new MyBlog
$ cd MyBlog
$ npx gulp server
```

### Development

We use https://lernajs.io

```sh
$ git clone # to nodos
$ cd nodos
$ make setup
$ make test
```
