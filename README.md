# Nodos

Node.js framework for humans

### Install

```sh
$ npm i -g @nodosjs/cli
$ nodos new MyBlog
$ cd MyBlog
$ bin/nodos server
# commands: bin/nodos --help
```

### Example

```sh
$ cd example
$ bin/nodos tests
$ bin/nodos server
```

### Development

We use https://lernajs.io

```sh
$ git clone # to nodos
$ cd nodos
$ make setup
$ make test
```

How to start (https://github.com/nodosjs/nodos/wiki). Russian version only.
